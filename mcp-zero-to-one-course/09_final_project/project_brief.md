# 期末專案：打造你的個人 AI 助理工具箱

恭喜你完成了 MCP 從零到一的學習！現在是展現實戰能力的時候了。

## 🎯 專案目標
開發一個整合性的 MCP Server，解決你日常生活或工作中的一個實際問題。

## 📝 需求清單
您的專案必須包含以下技術元素：
1.  **至少 2 個 Tools**：包含複雜邏輯與參數驗證。
2.  **至少 1 個 Resource**：提供一份動態或靜態的資料參考。
3.  **至少 1 個 Prompt 模板**：引導模型執行特定任務。
4.  **環境管理**：使用 `uv` 管理相依套件。
5.  **文件**：包含 `README.md`，說明如何啟動與測試。

## 💡 題目建議 (選一個或自訂)
- **個人理財助理**：讀取 CSV 帳單 (Resource)，提供記帳工具 (Tool)。
- **GitHub 專案管家**：讀取 Issue 列表，提供自動摘要與分類建議。
- **本地搜尋專家**：整合本地全文檢索工具，讓 Claude 能搜尋你的筆記。

## 🚀 提交檢查
- [ ] 程式碼可以透過 `uv run mcp dev` 成功啟動。
- [ ] 在 Claude Desktop 中能看到連線綠燈。
- [ ] 通過「安全性檢查清單」的基礎測試。
